# Logs

Logs are the digital footprint of activities in a computer system.

Logs are very important for an SOC analyst because they record events that occur across systems, applications, and network devices.

Examples:

- Device logs
- Firewall logs

Every activity or process generates logs that can be analyzed for suspicious behavior.

Logs are the "eyes and ears" of a system.

## Why Logs Matter

- Monitor activity
- Detect malicious behavior
- Perform forensic analysis
- Compliance and auditing

### Importance of Logs

- Incident detection
- Threat hunting
- Forensics
- Compliance
- Correlation
  - Combine logs from multiple sources using SIEM tools for detection and alerting.

# Types of Logs

## System Logs

- Boot events
- Shutdown events
- OS-related activities

## Security Logs

- Login attempts
- Malware detection
- Security events

## Application Logs

- Application errors
- User activity

## Firewall Logs

- Allowed traffic
- Blocked traffic

## Web Server Logs

- User-Agent

## Authentication Logs

- Generated by LDAP and other authentication services
- Successful logins
- Failed login attempts

## DNS Logs

- Domain name resolution

## Email Logs

- Spam emails
- Phishing emails
- Mail delivery events

## Proxy Logs

- User internet access
- Suspicious website visits

## Cloud Logs

- API calls
- Resource usage
- Cloud activity

## Endpoint logs 

- process creation
- file modifications
- registry changes (EDR)

# Common Log Formats

- **Syslog**
  - Standard logging format in UNIX/Linux systems.

- **JSON**
  - Structured key-value format.
  - Common in ELK Stack and modern logging platforms.

- **CEF (Common Event Format)**
  - Used by ArcSight.

- **LEEF (Log Event Extended Format)**
  - Used by IBM QRadar.

- **Plain Text / CSV**
  - Human-readable formats.
  - CSV stores comma-separated values.

# Common Mistakes

- Logs not enabled on endpoints
- Log retention period too short
- Log overload (collecting excessive logs)
- Not monitoring critical Event IDs
- Logs not protected against tampering
- Time synchronization issues (incorrect system clocks make investigations difficult)
